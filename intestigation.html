This analysis calculates Z-scores for PercentageWithin4HoursEpisode, helping us spot outliers which are data points that are significantly different from the average waiting time. Ive set a threshold at 3 standard deviations meaning that if a weeks Z-score exceeds 3 or falls below -3, its flagged as an outlier in the outliers_within4hours data frame. This helps us focus on weeks with unusual waiting time patterns.
